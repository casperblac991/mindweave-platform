## 🎯 خطوات النشر | Deployment Steps

### 1. الاختبار المحلي
```bash
python3 -m http.server 8000
```

### 2. النشر على GitHub Pages
```bash
git push origin main
# ثم فعّل GitHub Pages من الإعدادات
```

### 3. النشر على Vercel
```bash
vercel --prod
```

### 4. النشر على استضافة تقليدية
- رفع الملفات عبر FTP
- تكوين النطاق المخصص
- تفعيل HTTPS
