# 🧠 MindWeave Store

**🌐 Live Website:** [www.mindweave.store](https://www.mindweave.store)

## 📖 About the Project

**MindWeave** is a comprehensive global platform for AI prompts, digital templates, and creative tools. It is the first Arabic platform dedicated to empowering creators and AI enthusiasts by providing high-quality digital products, an interactive Prompt Lab, and a creator program that allows users to monetize their work.

---

## ✨ Key Features

| Feature | Description |
|--------|-------------|
| 🌍 **Multi-Language** | Arabic, English, Spanish, French (RTL/LTR support) |
| ⚗️ **Prompt Lab** | Generate, improve, and translate AI prompts for free |
| 🛒 **Digital Store** | 500+ digital products including AI prompts, Notion templates, design assets, e-books, and coding resources |
| 💰 **Creator Program** | Sell your digital products and earn up to 70% commission |
| 📱 **Responsive Design** | Fully optimized for desktop, tablet, and mobile devices |
| 🔒 **Secure Payments** | 100% secure checkout with instant delivery |
| 📧 **Newsletter** | Free 100 AI prompts on subscription |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Fonts:** Google Fonts (Cairo, Inter, Space Mono)
- **Monetization:** Google AdSense
- **Storage:** Local Storage (cart, language preferences)
