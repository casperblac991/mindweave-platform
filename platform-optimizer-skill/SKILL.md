# Platform Optimizer Skill

This skill provides a structured workflow to analyze an existing digital platform and develop a comprehensive strategy for its improvement and scaling. It addresses common challenges such as lack of customers, insufficient product offerings, outdated content, and ineffective marketing.

## Workflow

The optimization process follows these sequential steps. For each step, use the corresponding template from the `templates/` directory to structure your output.

1.  **Initial Platform Analysis**: Conduct a thorough analysis of the target platform's URL. Examine its design, user experience (UX/UI), technical architecture, features, performance, and SEO.
    *   **Action**: Navigate to the URL, analyze the HTML and visible content, and create an initial analysis report using `templates/1_platform_analysis_template.md`.

2.  **Product Expansion Strategy**: Develop a strategy to expand the platform's product or service offerings. Generate ideas for new digital products, features, or content that align with the platform's niche.
    *   **Action**: Generate a document outlining new product ideas using `templates/2_product_expansion_template.md`.

3.  **Content & SEO Strategy**: Create a content strategy for the platform's blog or content hub, focusing on SEO, keyword research, and content pillars to attract organic traffic.
    *   **Action**: Produce a content strategy document using `templates/3_content_strategy_template.md`.

4.  **Social Media Promotion Strategy**: Formulate a tailored social media plan to increase brand awareness and drive traffic. Define content types and engagement tactics for relevant platforms (e.g., X, Instagram, LinkedIn).
    *   **Action**: Deliver a social media strategy using `templates/4_social_media_strategy_template.md`.

5.  **Conversion Rate Optimization (CRO) Strategy**: Propose technical and design enhancements to improve conversion rates. This includes optimizing page speed, mobile experience, and the user's path to purchase.
    *   **Action**: Generate a CRO plan using `templates/5_platform_improvements_template.md`.

## Usage

To initiate the process, confirm the user's goal to analyze and improve a digital platform. Then, follow the workflow step-by-step, informing the user as you complete each stage and delivering the corresponding document.

**Example User Request:** "I have a website, mindweave.store, but I'm not getting any customers. Can you help me fix it and make it a global success?"

Begin with Step 1 and proceed sequentially through the other steps, using the provided templates to create the deliverables.
